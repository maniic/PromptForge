         - Returns `{ displayed: string, done: boolean }`
         - `useEffect` with `[text]` dependency — clears interval on cleanup
         - Increments character index via setInterval, calls onSeventyFivePercent at 75% completion
       - Also export a `TypewriterText` component for convenience:
         - Props: `{ text: string; speedMs?: number; onSeventyFivePercent?: () => void; className?: string }`
         - Renders `<span>{displayed}<span className="animate-pulse">|</span></span>` (blinking cursor while typing, cursor disappears when done)

    2. **InputCard.tsx** — Column 1 (user's original input):
       - Props: `{ inputText: string; onReset: () => void }`
       - Uses shadcn `Card` with custom styling: rounded-xl (12px), opaque bg one shade lighter than page, faint accent-colored outer glow/shadow (per locked decision — input card distinction)
       - Header: "Your Input" label + "New Forge" button (small, ghost variant, with RotateCcw icon from Lucide)
       - Body: displays `inputText` in a readable format (whitespace-pre-wrap)
       - "New Forge" click calls `onReset` which animates back to hero

    3. **CraftedCard.tsx** — Column 2 (crafted prompt with typewriter):
       - Props: `{ craftedPrompt: string; onSeventyFivePercent: () => void }`
       - Uses shadcn `Card`, neutral dark shadow (no accent glow — per locked decision, result cards neutral)
       - Header: "Crafted Prompt" label + category badge (passed as prop or derive from parent)
       - Body: `<TypewriterText text={craftedPrompt} speedMs={15} onSeventyFivePercent={onSeventyFivePercent} />`
       - Text styled with `text-sm leading-relaxed whitespace-pre-wrap`

    4. **ResultCard.tsx** — Column 3 (before/after comparison):
       - Props: `{ rawResult: string; craftedResult: string; visible: boolean }`
       - Uses shadcn `Card`, neutral dark shadow
       - Two sections stacked vertically:
         - "Raw Result" (labeled "Before" or "Without PromptForge"): displays rawResult with muted styling
         - "Crafted Result" (labeled "After" or "With PromptForge"): displays craftedResult with accent-highlighted styling
       - Each section in its own sub-card or bordered area for visual separation
       - `visible` prop controls opacity via motion: `animate={{ opacity: visible ? 1 : 0 }}` with 500ms transition
       - Text: `text-sm leading-relaxed whitespace-pre-wrap` for both

    5. **ThreeColumnLayout.tsx** — Per research Pattern 2:
       - Props: `{ result: ForgeResponse; inputText: string; onReset: () => void }`
       - Local state: `showResults` boolean, starts false, set to true when typewriter hits 75%
       - Layout: CSS grid, `grid-cols-1 lg:grid-cols-[1fr_1.2fr_1.2fr]` (input slightly narrower), `gap-5`
       - Column 1: `<InputCard inputText={inputText} onReset={onReset} />` — no motion wrapper needed (it slides in from the hero transition)
       - Column 2: wrapped in `<motion.div>` with staggered slide-in per COLUMN_VARIANTS:
         - `initial={{ opacity: 0, x: 60 }}`, `animate={{ opacity: 1, x: 0 }}`, `transition={{ duration: 0.35, delay: 0, ease: 'easeOut' }}`
         - Contains `<CraftedCard craftedPrompt={result.crafted_prompt} onSeventyFivePercent={() => setShowResults(true)} />`
       - Column 3: wrapped in `<motion.div>` with stagger:
         - `initial={{ opacity: 0, x: 60 }}`, `animate={{ opacity: 1, x: 0 }}`, `transition={{ duration: 0.35, delay: 0.15, ease: 'easeOut' }}`
         - Contains `<ResultCard rawResult={result.raw_result} craftedResult={result.crafted_result} visible={showResults} />`
       - Full width container with horizontal padding: `max-w-7xl mx-auto px-4`
