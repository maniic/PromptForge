# PromptForge — Product specification

## Problem
1.8 billion people use AI tools. Most get mediocre results because
they don't know how to prompt. The gap between what people want
and what they can articulate is the last-mile problem of AI adoption.
This affects everyone: beginners, students, professionals, seniors.

## Solution
PromptForge uses IBM Granite to bridge that gap.
Type any rough idea → Granite engineers the perfect prompt →
Granite executes it → you see the result alongside what your
bad prompt would have produced. First-time users get expert-level
output on their very first try.

## Core user journey
1. User arrives at dashboard — sees 3 demo preset buttons
2. Clicks "Vibe code: make me a habit tracker app"
3. Granite detects category: vibe_coding
4. Center panel: crafted prompt types out character by character
5. Right panel splits: green (good) vs red (bad) results appear
6. Difference is immediately visible — same model, different prompt
7. User clicks "Save to Library" → community prompt added

## Categories
- Vibe coding: natural language → working app/code
- Brainstorming: vague idea → structured plans and ideas
- Q&A / research: rough question → comprehensive answer
- One-shot: any task → single perfectly structured prompt

## Community library
Public prompt library indexed by category.
Browse best prompts → one click to use → upvote the best ones.
Network effects: community collectively gets better at prompting.
